Compact status/label pill. Four roles cover every tag in the archive.

```jsx
<Badge variant="status" dot>Verified</Badge>
<Badge variant="info" mono>opus-4.8</Badge>
<Badge variant="neutral" mono>fable-5</Badge>
<Badge variant="alert">502</Badge>
```

`variant`: neutral · info · status · alert. `dot` adds a leading status dot (emerald on status). `mono` switches to JetBrains Mono micro size for model chips / IDs. `icon` for a leading glyph.
