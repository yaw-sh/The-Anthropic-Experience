The surface primitive: a white, hairline-bordered, softly-shadowed rounded container.

```jsx
<Card>Standard raised content.</Card>
<Card variant="muted">Live environment numbers summary strip.</Card>
<Card variant="flat" padding="var(--space-3)">Nested inline block.</Card>
```

`variant`: default · muted (zinc wash) · flat (no shadow). `padding` overrides the default 24px.
