# Evidence Inputs

This directory is for user-approved, sanitized evidence used by the public experience.

## First case source

Before Phase 2, place the approved reconstructed transcript at:

```text
docs/evidence/chattranscript-2026-07-18-sanitized.md
```

The source must retain a visible fidelity note stating that it was reconstructed and sanitized by the assistant rather than exported byte-for-byte from the platform log.

## Public-content rule

Do not commit:

- Unredacted chat exports
- Names, account identifiers, private repository names, machine paths, credentials, or tokens
- Personal, family, health, education-record, or professional information not required for the case
- Screenshots containing private account details

## Claim review

Every public claim must point to one source record and one evidence classification. When a source cannot establish a claim, use `unknown` or remove the claim.
