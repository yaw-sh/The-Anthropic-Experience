# Wheel prototype v1

This directory retains the sanitized implementation files unique to the larger first wheel archive plus the exact scaffold files shared by both unique archives. Remote URLs and private reference slots are sanitized. The files are historical visual/reference material and are not the canonical application.

The shared scaffold moved here during product integration because `app/` now contains the canonical TypeScript product rather than the historical wheel prototype. Shared files are retained once; generated dependencies and the ZIP container remain removed.
