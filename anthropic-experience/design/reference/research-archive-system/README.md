# Research archive design reference

This public-safe reference retains the unique reusable source from the delivered research-archive design system: generic React components, type declarations, design tokens, and the CSS import manifest.

Raw transcript previews, sample conversation text, generated bundles, remote preview pages, thumbnails, prompt examples, and compiled support files were removed. Remote font URLs were replaced with `###-PII-###`; the reference therefore requires local font assets before use.

This directory is historical design reference only. It is not a product implementation and does not supersede `app/` or the root master blueprint.
