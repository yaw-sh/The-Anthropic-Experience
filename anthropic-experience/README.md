# THE ANTHROPIC EXPERIENCE

This repository is the canonical product and its sanitized public process record.

- [`app/`](app/) is the canonical browser build.
- [`THE-ANTHROPIC-EXPERIENCE-MASTER-BLUEPRINT.md`](THE-ANTHROPIC-EXPERIENCE-MASTER-BLUEPRINT.md) is the authoritative 2,133-line creative and technical blueprint.
- [`history/`](history/) presents six chronological phases with generalized transcripts and source-range receipts.
- [`evidence/public/`](evidence/public/) provides the stable JSON interface consumed by the product.
- [`docs/audit/`](docs/audit/) contains only public-safe cleanup, deduplication, transcript, redaction, and integration receipts.
- [`design/`](design/) contains approved scene art and canonicalized public-safe design references.
- [`design/reference/claude-artifact/`](design/reference/claude-artifact/) preserves the supplied Claude artifact as a byte-identical, non-runtime design reference.

The published process does not include raw/private sibling editions. Operator, assistant, system, and tool content are generalized; private reference slots use the literal `###-PII-###`; duplicate copies and ZIP archives are removed.

The product-build and repository-cleanup tracks remain separately visible until integration. A branch push does not establish completion: the product, sanitized content, and verified rewritten history must be integrated and visible on `main`.
